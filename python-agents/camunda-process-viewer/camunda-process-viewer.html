Camunda 8.8 Process Instance Viewer

Setup Instructions

To use this process viewer, run the server first:

  1. Open a terminal in this directory
  2. Run: python camunda-process-viewer-server.py or double-click start-camunda-process-viewer-server.bat
  3. Then access this page via: http://localhost:3001/camunda-process-viewer.html

Note: Make sure Camunda is running on port 8088
